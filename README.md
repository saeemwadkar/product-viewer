# Product Viewer App

## Tech Stack
- React 18+
- React Router 6
- Tailwind CSS
- Axios
- Lucide React (icons)
- react-hot-toast (notifications)
- DummyJSON API for auth + products

## Features
✅ Login with DummyJSON  
✅ JWT stored in localStorage  
✅ Protected product listing  
✅ Token expiry check  
✅ Logout  
✅ Search / sort products  

## How to run
```bash
npm install
npm start
